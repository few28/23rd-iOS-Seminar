## 01. 상수와 변수
### 상수, 변수의 선언
* 상수 선언 키워드 `let`
* 변수 선언 키워드 `var`  
```swift
let name: Type = value  
var name: Type = value  
```
> 값의 타입이 명확하다면 타입 생략 가능
```swift
let name = value
var name = value
```
```swift
let constant: String = "선언 후 값의 변경이 불가능한 상수"
var variable: String = "선언 후에도 변경이 가능한 변수"

variable = "변수는 이렇게 다른 값을 할당할 수 있지만"
constant = "상수는 불가능합니다." // 에러
```
